Instrucciones para Reiniciar la Base de Datos

Este documento explica cómo reiniciar la base de datos a su estado inicial. Este proceso borrará todos los datos y creará un usuario administrador por defecto.

---

### Usuario Administrador por Defecto:

*   **Email**: `admin_general@sti.com`
*   **Contraseña**: `Admin01`

---

### Pasos para Reiniciar la Base de Datos:

1.  Abre una terminal en la raíz del proyecto.
2.  Ejecuta el siguiente comando:

    ```bash
    npm run db:reset
    ```

### ¿Qué hace este comando?

El comando `npm run db:reset` realiza las siguientes acciones:

1.  **Borra la base de datos** actual.
2.  **Aplica las migraciones** para crear la estructura de la base de datos.
3.  **Ejecuta el script de seeding** (`prisma/seed.js`) para crear el usuario administrador.

Este comando es útil durante el desarrollo para empezar con una base de datos limpia y consistente.
